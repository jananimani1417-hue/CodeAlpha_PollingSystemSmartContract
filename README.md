# CodeAlpha_PollingSystemSmartContract
A decentralized polling system built with Solidity that allows users to create polls, cast votes, enforce voting deadlines, prevent duplicate voting, and determine winning options securely using blockchain technology.  
Project Overview

This project is a blockchain-based polling system that enables secure and transparent voting through smart contracts.

Features
Create polls with title and options
Set voting deadline
One vote per address
Prevent duplicate voting
Store votes securely using mappings
Calculate and display winning option
Technologies Used
Solidity
Ethereum
Remix IDE

Outcome

Successfully created and tested a decentralized voting system with secure vote management.
